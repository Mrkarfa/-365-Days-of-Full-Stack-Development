**TechSummit 2026** is a comprehensive event website template featuring an extensive form system (Registration, Login, Feedback) and a modern, high-contrast dark theme design. It demonstrates advanced HTML5 form elements styled with CSS3 variables for a cohesive user interface.

- **Theme**: Cyber/Tech Dark Mode.

- **Visuals**:

  - Vibrant Gradients (Blue to Violet) for Call-to-Actions and headings.
  - Glassmorphism effects on the sticky navigation bar.

## âœ¨ Key Features

### 1. Advanced HTML Forms

The project showcases varying complex form scenarios:

- **Registration Form**:
  - Grouped fields using `<fieldset>` and `<legend>`.
  - Semantic inputs: `date`, `tel`, `email`, `url`.
  - Dropdowns (`<select>`) with `<optgroup>`.
- **Login System**:
  - Focus states for better accessibility.
  - "Remember Me" checkbox styling.
- **Feedback Survey**:
  - Range sliders (`<input type="range">`) for NPS scores.
  - Radio button groups for Likert scale ratings.

### 2. CSS Architecture

- **CSS Variables**: Used for colors, spacing, and fonts to ensure consistency and easy theming.
- **Responsiveness**:
  - Mobile-first approach.
  - Full-width inputs on smaller screens.
  - Stacked navigation for mobile users.
- **Interactivity**:
  - Hover effects on cards and buttons.
  - Focus rings for keyboard navigation accessibility.
